# 延迟加载状态

有些时候，当请求返回足够快的情况下，loading 可能会在短时间内完成 false -> true -> false 状态的切换。这时候，加载动画可能会出现闪烁的情况（特别是占满一整屏的动画），这给会用户带来糟糕的体验。

因此最好的方法就是设定一个延迟值，当等待时间大于延迟值时 loading 才会被设置成 true，如下所示

import { defineComponent } from 'vue';
import { useRequest } from 'vue-request';

function testService(): Promise<number> {
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve(new Date().getTime());
    }, 100);
  });
}

export default defineComponent({
  name: 'App',
  setup() {
    const { data: delayData, loading: delayLoading, run: delayRun } = useRequest(testService, {
      loadingDelay: 500,
    });
    const { data, loading, run } = useRequest(testService);
    return () => (
      <div>
        <button
          onClick={() => {
            run();
            delayRun();
          }}
        >
          run
        </button>
        <br />
        <br />
        <div>Current Time (default): {loading.value ? 'loading...' : data.value}</div>
        <div>Current Time (delay): {delayLoading.value ? 'loading...' : delayData.value}</div>
      </div>
    );
  },
});















 




















当然，世事无绝对，假如我们把 loadingDelay 设置成了 500ms ，但实际上我的请求在 510ms 完成了，这时还是无法避免会出现闪烁。
